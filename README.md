\#  Decoding the Basket  

\### Customer Segmentation \& Market Basket Analysis using Python and Power BI



This project focuses on analyzing customer purchasing behavior using \*\*Market Basket Analysis\*\* and presenting actionable insights through an \*\*interactive Power BI dashboard\*\*. The goal is to uncover frequently bought product combinations and support data-driven cross-selling and recommendation strategies.



---



\##  Project Objectives



\- Understand customer purchase patterns

\- Identify frequently bought together products

\- Generate association rules using Apriori algorithm

\- Visualize insights through an interactive dashboard

\- Provide business-ready recommendations



---



\##  Tools \& Technologies Used



\- \*\*Python\*\*: Data preprocessing, analysis  

&nbsp; - pandas, numpy  

&nbsp; - mlxtend (Apriori \& Association Rules)

\- \*\*Power BI\*\*: Dashboard creation \& visualization

\- \*\*Git \& GitHub\*\*: Version control

\- \*\*Jupyter Notebook\*\*: Analysis workflow

\##  Key Analysis Steps



1\. Data cleaning and preprocessing

2\. Transaction-level basket creation

3\. Frequent itemset generation using Apriori algorithm

4\. Association rule mining using support, confidence, and lift

5\. Exporting clean tables for Power BI

6\. Dashboard creation and visual analysis
